Project Description

Our project is a Lost and Found website for the University of Nevada Reno. On this website, the user will be shown a map with markers for each building on campus, and for each building, the user will be able to click on them and a list of all items on that lost and found will be displayed. The user can also check the items that were claimed on that building. Any student workers will be able to log in to the website to log items for the buildings and remove items from buildings. Admins can also log in to validate student accounts and SuperAdmins can make new admin and student accounts (as well as any other functionalities the other roles can do.)
